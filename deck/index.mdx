### What is a React Component?

- React Component is a JavaScript function that returns html.

```jsx
function HelloWorld() {
  return <h1>Hello World</h1>;
}

// to render a react component
ReactDOM.render(<HelloWorld />, document.getElementById("root"));
```

### How are we exposing the React Component to JSP?

```jsx
function HelloWorld(props) {
  return (
    <div>
      <h1>Hello World</h1>
      {/* a component can nest other components*/}
    </div>
  );
}
// by creating a function on the window object we are exposing the react component to JSP
// You can call this function from your jsp page ðŸ‘‡
// renderPreviousVehicles("idofthediv", {data: {}, someFunction: function() {}})
window.renderHelloWorld = function renderHelloWorld(mountId, addInfo) {
  ReactDOM.render(
    <HelloWorld addInfo={addInfo} />,
    document.getElementById(mountId)
  );
};
```

### How are we using React on our project?

- We are not building the whole page in React(that's the traditional way).
- We are identifying components on the page and converting one or two components into React and integrating them back to the page.
- So our page will contain both legacy components(jQuery + HTML) and the new React components and we need to make sure that the legacy and new React components can talk to each other.
